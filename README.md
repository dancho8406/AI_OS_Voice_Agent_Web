# AI OS Voice Agent - Web Cloud Edition

Гласов асистент, базиран на уеб технологии, проектиран с клиент-сървър архитектура и деплойнат изцяло в облачна среда.

## 🔗 Линкове към проекта
* **Работещо приложение (Фронтенд във Vercel):** https://frontend-eta-ecru-85.vercel.app/
* **Облачен бекенд API (Google Cloud Run):** https://web-agent-896874138272.europe-west4.run.app

## 🏗️ Архитектура на системата
Проектът е изграден от два независими компонента:
1. **Клиентска част (Frontend):** Разработена с React, Vite и TypeScript. Използва вградената в браузъра Web Speech API (`webkitSpeechRecognition`) за обработка на гласови команди на български език. Деплойната е в платформата Vercel.
2. **Сървърна част (Backend):** Изградена с FastAPI (Python). Анализира изпратените команди, обработва съвпаденията и връща структуриран JSON отговор към клиента. Деплойната е като контейнеризирана услуга (Docker) в Google Cloud Run.

## 🚀 Функционалности
* Гласово разпознаване в реално време на български и английски език.
* Автоматично отваряне на популярни уебсайтове (YouTube, Facebook, Google, ABV, Синоптик, bTV и др.) чрез гласова команда.
* Заобикаляне на CORS ограниченията чрез специализирани CORS Headers сигурностни политики в FastAPI.
